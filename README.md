# Personal Website Project

Features in Website 
- Interactive portfolio showcasing my coding projects and simulations.
- Embedded YouTube video demonstrating my work and explanations about my coding.
- Project timeline with completed and upcoming projects.
- Reponsive design for different screen sizes.
- Smooth animations using @keyframes for titles, links, and sections.
- Includes social accounts links to GitHub, YouTube, and Linktree.

How it Works
- The website is structured by HTML elements for layout.
- CSS is used for the animations and styles within the website.
- Sections are seperate into project, videos, and socials.
- Content will be updated as new projects are completed over time.
- YouTube videos are embedded using an iframe.
- Users can create connections with my projects and potentially gain motivation.

Requirements
- IDE (e.g., VS Code, etc).
- HTML5 (structuring of the web page).
- CSS3 (used for styling, animation, and layout).
- Modern Web Browser (Chrome, Edge, Safari).
- Image files:
  1) icon.jpg

How to Run
1. Copy the HTML code into a HTML file named "index.html".
2. Copy the CSS code into a CSS file named "style.css".
3. Make sure the icon file is in the same directory with CSS and HTML file, or else the icon won't appear in the tab of website.
4. Make sure to have a working extension that will launch a local development server with a live reload feature for both static and dynamic web pages (e.g., Live Server, etc).
